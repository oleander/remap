Overview

Represents a non-empty rule block

Examples:

A non-empty collection

class Mapper < Remap::Base
  define do
    map do
      map :a1, to: :b1
      map :a2, to: :b2
    end
  end
end

Mapper.call({ a1: 1, a2: 2 }).result # => { b1: 1, b2: 2 }
